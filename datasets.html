We created five different layers with objects:

  • Clubs layer
  • Pubs layer
  • Restaurants layer
  • Universities layer
  • Poznan districts layer

Every layer has been created in Qgis Brighton 2.6 using OSM as basemap. After that, vector layers have been exported to .geoJSON files. From every single .geoJSON file we extracted "coordinates" and "name" attributes of points and added it to javaScript arrays. Thanks to the javaScripts arrays we could use markercluster.js plugin and leaflet.js library to add our points on map in that fancy way.
